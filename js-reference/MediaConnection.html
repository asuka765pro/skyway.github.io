Class: MediaConnection

MediaConnection(remoteId, optionsopt)

Class that manages data connections to other peers.

Constructor

new MediaConnection(remoteId, optionsopt)

Create a data connection to another peer.
Parameters:
Name Type Attributes Description
remoteId string The peerId of the peer you are connecting to.
options object <optional>
 Optional arguments for the connection.
Properties
Name Type Attributes Description
connectionId string <optional>
 An ID to uniquely identify the connection. Defaults to random string if not specified.
label string <optional>
 Label to easily identify the connection on either peer.
pcConfig object <optional>
 A RTCConfiguration dictionary for the RTCPeerConnection.
stream object <optional>
 The MediaStream to send to the remote peer. Set only when on the caller side.
originator boolean <optional>
 true means the peer is the originator of the connection.
queuedMessages string <optional>
 An array of messages that were already received before the connection was created.
payload string <optional>
 An offer message that triggered creating this object.
videoBandwidth number <optional>
 A max video bandwidth(kbps)
audioBandwidth number <optional>
 A max audio bandwidth(kbps)
videoCodec string <optional>
 A video codec like 'H264'
audioCodec string <optional>
 A video codec like 'PCMU'
