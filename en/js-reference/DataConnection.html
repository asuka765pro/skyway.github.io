Class: DataConnection

DataConnection(remoteId, optionsopt)

Class that manages data connections to other peers.

Constructor

new DataConnection(remoteId, optionsopt)

Create a data connection to another peer.
Parameters:
Name Type Attributes Description
remoteId string The peerId of the peer you are connecting to.
options object <optional>
 Optional arguments for the connection.
Properties
Name Type Attributes Description
connectionId string <optional>
 An ID to uniquely identify the connection. Defaults to random string if not specified.
serialization string <optional>
 How to serialize data when sending. One of 'binary', 'json' or 'none'.
label string <optional>
 Label to easily identify the connection on either peer.
queuedMessages string <optional>
 An array of messages that were already received before the connection was created.
payload string <optional>
 An offer message that triggered creating this object.
